# Bus Timetables

An example of translations in Java. Has translations for `en-AU` (Australian
English) and `en-PT` (Pirate speak).

## Compile & Run

### Use default locale

```sh
./gradlew run
```

### Specify a locale

```sh
./gradlew run --args='--locale=en-AU'
```

or

```sh
./gradlew run --args='--locale=en-PT'
```
